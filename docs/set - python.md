# Set
- 중복을 허용하지 않고 순서가 없는 자료형임
- 이건 진짜로 순서가 없다는거~
- 인덱싱 지원하지 않는다.
- C-family와는 다르게 진짜 math-set 연산 (`intersection`: `&`, `union`: `|`, `difference`: `-`, `symmetric_difference`: `^`)들을 지원한다.
- `in` 문법을 사용하여 해당 값이 집합 안에 존재하는지 질의할 수 있다.
