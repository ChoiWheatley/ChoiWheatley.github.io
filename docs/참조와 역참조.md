```mermaid
flowchart LR

Post -- 참조 --> Comment
Comment -- 역참조 --> Post
```

- 참조: `comment.post.id`
- 역참조: `post.comment_set.all()`
	- ==`_set`==에 유의할 것
- Foreign Key를 Comment가 가지고 있기 때문에 Comment의 객체를 사용하여 Post를 참조할 수 있다는 것은 잘 알고 있는 사실이다.
- Post를 가지고도 거꾸로 참조관계를 추적할 수 있지롱~ Post 객체 하나에 여러 Comment 객체가 달릴 수 있으니까 `QuerySet`을 요청할 수 있다.
