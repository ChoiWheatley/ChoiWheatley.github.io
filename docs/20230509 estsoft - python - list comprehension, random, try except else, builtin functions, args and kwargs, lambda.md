# List Comprehension

`for` + `append` 보다 속도도 빠르고 가독성도 좋은데 안 쓸이유가 없잖아?

구문:
```python
[{{expr}} for {{variable}} in {{iterable}} {{condition}}]
```

- condition쪽은 옵션이다. 필터링 할 때 아주 유용함.
