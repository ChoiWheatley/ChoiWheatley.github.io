- css에서 `margin: auto`를 주면 해당 축으로 부모요소의 정 중앙에 디비잡게 된다.
- css `display: inline-block` 하게 되면 블럭속성을 가진 요소를 좌,우로 병치할 수 있게 된다. 아이디, 비밀번호가 `<label>` 태그인데, 저렇게 속성을 주어서 저시기 되었다.
