## TL;DR

단위 테스트를 수행할 때 자주 사용되는 패턴으로 아래 세 가지를 지키면 되는 간단한 규칙이다.

- Arrange: 객체를 생성하고 적절하게 초기화하라.
- Act: 객체에 테스트 동작을 수행시켜라
- Assert: 예상된 결과를 평가하라.
