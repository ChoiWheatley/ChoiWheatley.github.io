- <https://brew.sh>

install homebrew: 아래 명령줄을 복사하여 터미널에 복붙하면 된다.

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

Apple Silicon Mac에서 Homebrew를 설치하면 이후에 명령어가 하나 뜨게 된다. 놓치지 말고 따라하자.
