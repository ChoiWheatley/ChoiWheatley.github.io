DNS 서버는 IP와 도메인 네임을 매핑해주는 서비스를 제공합니다.

> 전 세계에 있는 모든 도메인 네임을 관리하기 위해선 어떻게 해야 할까요?

분산된 도메인 네임 시스템이라는 기술을 사용합니다. 계층구조로 이루어져 있기에 사용자가 도메인 네임을 검색할때 여러 DNS 서버를 통해 목적지 주소를 찾아내게 됩니다.
