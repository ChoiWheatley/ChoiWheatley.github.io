    README

    핸드폰으로 구글킵에 싸질러놓은 아이디어들과 링크를 1차적으로 계류시키는 장소이다. 여기에서 파생된 아이디어는 새로운 인덱스로 갈 수도 있고 그대로 묻혀버릴 수도 있는거고. 그건 나의 관심에 따라 흘러가겠고, 나는 그런 것들에 대한 정렬을 주기적으로 진행할 것이다.
