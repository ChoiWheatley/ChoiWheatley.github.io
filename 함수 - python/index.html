    What is Closure?¶

    아래는 제곱을 수행하는 또하나의 방법을 표현한다. 이것은 팩토리 함수라고도 부른다. 원래는 휘발되었어야 하는 x를 가진 채 x ** y를 수행하는 함수를 리턴할 수 있다.

    def 제곱(x):
    def 승수(y):
        return x**y

    return 승수
